PydanticAI Deep-Research Agent

A minimal research agent with two tools, a human-in-the-loop approval gate, structured output, and OpenTelemetry tracing into Arize Phoenix.

Agent execution flow

The model decides which tool to call. search_web runs immediately. query_internal_db is registered with requires_approval=True, so PydanticAI returns a DeferredToolRequests and the runtime asks a human. The loop continues until the model emits a FinalAnswer or the tool_calls_limit is hit.

Observability pipeline

Agent(..., instrument=True) makes PydanticAI emit OpenTelemetry spans for every turn, tool call, and HITL gate. The spans are enhanced with OpenInference semantic attributes in-place, then shipped to a locally running Phoenix over OTLP/HTTP.
